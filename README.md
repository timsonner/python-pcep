# PCEP Exam Preparation (Python-PCEP)

This repository follows the **PCEP-30-02** syllabus. Each block is subdivided into folders with Python files containing learning objectives and tasks.

## Syllabus Reference
- [Official PCEP Syllabus](https://pythoninstitute.org/pcep-exam-syllabus)
- [Python 3.12 Documentation (Range)](https://docs.python.org/3.12/library/stdtypes.html#range)

## Course Structure

### Block 1: Computer Programming and Python Fundamentals
- 1.1: Fundamentals & Definitions
- 1.2: Logic & Structure
- 1.3: Literals, Variables & Numeral Systems
- 1.4: Operators & Data Types
- 1.5: Input/Output operations

### Block 2: Control Flow – Conditional Blocks and Loops
- 2.1: Decisions & Branching
- 2.2: Iterations (Loops)

### Block 3: Data Collections – Tuples, Dictionaries, Lists, and Strings
- 3.1: Lists
- 3.2: Tuples
- 3.3: Dictionaries
- 3.4: Strings

### Block 4: Functions and Exceptions
- 4.1: Functions (Decomposition)
- 4.2: Function Environment
- 4.3: Exceptions Hierarchy
- 4.4: Exception Handling
